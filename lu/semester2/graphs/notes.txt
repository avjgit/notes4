================== 1.1 Kā rodas grafi?
atslēgvārdi:
    virsotnes
    šķautnes

piemēri
    daudzskaldnu grafi

    ielu plani
    turniru grafi
    variantu koki

vingrinajums 1:
    volejbola turnīrs:
        A-> E, D
        B-> A, C, D
        C-> A
        D-> C, E
        E-> B, C

    kurai komandai..
    .. 1.vieta? => B (jo visvairak uzvaru)
    .. 5.vieta? => C (jo vismazak uzvaru)
    .. 2 punkti?=> A, D un E

    futbola turnīrs:
        A-> B, C, D
        B-> D
        C-> B
        D->
        E-> C

    kurai komdai..
    .. visvairak uzvaru? => A
    .. vismazak uzvaru?  => D
    .. nav neizskirtu?   => nav (katrai ir neizskirta)
